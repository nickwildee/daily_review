```
# 항, 인수, 계수

## 항이란?

덧셈(또는 뺄셈)으로 이루어진 식에서 덧셈 기호나 뺄셈 기호로 구분되는 각각의 부분을 항이라고 한다.

1. 3x + 5
=> 항은 3x, 5 (총 2항)
2. 7a - 2b + 4
=> 항은 7a, 2b, 4 (총 3항)
3. x^2 + 3x + 2
=> 항은 x^2, 3x, 2 (총 3항)

!주의사항

항 안에서는 곱셈/나눗셈/지수가 들어 있어도 하나의 항으로 본다.
덧셈/뺄셈 기호가 나올 때마다 항이 끊어진다고 생각하면 됨

ex) -2/3xy^2 => 이건 "하나의 항"임

## 인수란?
어떤 수나 식이 곱의 형태로 나타나 있을 때, 그 곱을 이루는 각각의 부분을 인수라고 한다.

1. 12 = 3 * 4
 => 3과 4는 12의 인수
2. 2xy
 => 인수는 2, x, y
3. (x - 1)(y - 2)
 => 인수는 (x - 1),(y - 2)
4. a^2b
 => 인수는 a, a, b (또는 a^2, b)

## 계수란?

미지수(문자) 앞에 곱해져 있는 수
항에서 숫자(상수 인수)가 따로 안보이면, 계수는 1이라고 본다.
만약 부호가 붙어 있으면 +1 또는 -1이 계수가 됨.
```

```
# 곱셈의 법칙 정리
## 곱셈의 교환법칙 & 결합법칙

교환법칙:
𝑎 × 𝑏 = 𝑏 × 𝑎

결합법칙:
(𝑎 × 𝑏) × 𝑐 = 𝑎 × (𝑏 × 𝑐)

덧셈의 교환·결합법칙과 동일하다.

!단, 항끼리 덧셈/뺄셈으로 연결된 경우에는 직접 적용되지 않는다.

## 곱셈의 분배법칙

곱셈은 덧셈/뺄셈에 대해 분배된다.

𝑎 × (𝑏 + 𝑐) = 𝑎 × 𝑏 + 𝑎 × 𝑐
𝑎 × (𝑏 − 𝑐) = 𝑎 × 𝑏 − 𝑎 × 𝑐

 전개(Expanding)

곱꼴을 덧셈꼴로 바꾸는 것

𝑎(2 + 4) = 2𝑎 + 4𝑎

 인수분해(Factoring)

덧셈꼴을 공통인수로 묶어 곱꼴로 바꾸는 것

2𝑎 + 4𝑎 = 𝑎(2 + 4)
```

```
동류항의 구분

계수를 제외한 미지수 부분(문자)가 지수까지 완전히 동일한 경우를 뜻한다.

동류항끼리의 덧셈/뺄셈 연산은 계수끼리의 연산만 진행 후 문자 부분은 건드리지 않는다.
```
