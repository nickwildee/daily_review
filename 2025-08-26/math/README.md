# [수학] 학습 회고 (2025-08-26)

## 학습한 내용

1. 양수와 음수를 이용한 사칙 연산

```
-1 + (-2)^2 * 1/3 = -1 + 4/3 = 1/3

2 - (2^2) + 4 * (-1/4) =
2 - 4 - 1 = -3
```

2. 정수(양,0,음)과, 자연수, 유리수와 무리수

```
유리수란 분수로 나타낼 수 있는 수,
무리수는 유리수가 아닌 실수.

정수란 소수점이 없는 유리수를 뜻한다.
(0.3 = 3/10) // 소수점이 포함된 유리수 => 정수가 아님
(1 = 1/1) // 소수점이 포함되지 않은 유리수 => 정수

양과 음의 정수란, 각각 양수인 정수, 음수인 정수를 뜻하며,
0은 양쪽 모두에 속하지 않은 정수이다.

자연수란 1부터 시작하는 양의 정수이다.
```
