## 학습한 내용

복소수의 연산, 켤레복소수

```
켤레복소수란 원래 복소수의 허수부의 부호만 다른 복소수를 뜻한다.
ex)

3 + i의 켤레복소수 = 3 - i
-i + 2의 켤레복소수 = i + 2

실수의 켤레복소수는 자기 자신과 같다.

켤레복소수끼리의 덧셈 = 실수부 * 2
        //      뺄셈 = 허수부 * 2
        //      곱셈 = 실수부 + 허수부 // 항상 실수가 됨
```

```
복소수의 덧셈, 뺄셈

다항식의 덧셈 뺄셈과 같이 동류항끼리 덧셈 뺄셈을 진행한다.

ex)
(3 - 2i) + (-2 + i) = 1 - i
(-2 + 5i) - (-3 - 2i) = 1 + 7i
```

```
복소수의 곱셈

전개해서 풀이할 수도 있지만 실수부가 되는 부분 + 허수부가 되는 부분으로 계산하면 더 빠르다.

ex)
(3 - 2i)(4 + i) = 14 - 5i
실수부가 되는 부분 = (3 * 4) + -2i * i = 12 + 2 = 14
허수부가 되는 부분 = (-2i * 4) + (3 * i) = -8i + 3i = -5i

합차공식과 제곱공식을 이용할 수 있다.

ex) 합차공식
(3 - 2i)(3 + 2i) = (a^2) - (b^2)
a(3)^2 - b(2i)^2 =
9 - (2i^2 = 4i^2 = 4 * -1 = -4) =
9 + 4 =
13

ex) 제곱공식
(3 + 2i)^2 = a^2 + 2ab + b^2 =
3^2 + (2 * 3 * 2i) + 2i^2 =
9 + 12i -4 =
5 + 12i
```

```
복소수의 나눗셈

분모의 실수화를 통해 풀이할 수 있다.
복소수는 켤레복소수와 곱하면 실수가 된다.
분자 분모에 각각 분모의 켤레복소수를 곱해준다.

ex)
2 - 5i / 4 + 3i =
(2-5i)(4-3i) / 25 =
-7 - 26i / 25 =
-7/25 - 26/25i

```
