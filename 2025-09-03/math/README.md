1. 유한소수, 무한소수, 순환소수

```
# 유한소수, 무한소수
소수는 유한소수와 무한소수로 나뉘어진다.

유한 소수는 소수점 아래가 유한한 소수를 말한다.
유한 소수는 모두 분수로 나타낼 수 있는 유리수이다.
ex) 1.1  = 11/10 , 3.15 = 315/100

무한 소수는 소수점 아래가 무한한 소수를 말한다.
무한 소수는 두가지이며 순환소수와 비순환 소수로 나뉘어진다.

순환 소수는 일정한 숫자 패턴이 반복되는 소수이다.
이들은 모두 유리수이다.
ex) 0.33333333... = 1/3 , 0.142857142857... = 1/7

비순환소수는 규칙적으로 반복되지 않고 무한히 이어지는 소수를 뜻한다.
ex) π=3.14159265...

```

2. 유한소수의 판별법

```
기약분수 𝑝/𝑞(𝑝,𝑞는 서로소, 𝑞>0)일 때,
그 분수가 유한소수로 표현되려면 분모 𝑞의 소인수가 2나 5뿐이어야 한다.

소수는 10진법으로 나타내기 때문에 분모를 10^k = 2^k * 5^k, 꼴로 바꿀 수 있어야 한다.
분모가 2와 5로만 이루어져 있다면, 적당한 k를 곱해서 분모를 10^k꼴로 만들 수 있음.
그럼 결국 소수점 아래 k자리에서 딱 끊기는 유한소수가 됨.
```

3. 순환소수를 분수로 표현하는 법

```
x = 5.033333...

정수 부분: 5

소수 부분:
0.0333...
0.0333...

순환마디: 3 (소숫점 이하의 "3"만 무한 반복)

순환마디가 소숫점 아래 첫째 자리부터 시작하므로,
x에 10을 곱하면 10x = 50.33333...

순환마디가 한 자리라서, 10배 더 곱한 100x를 만든다:
100x = 503.3333...

뺄셈으로 순환 부분 제거
100x - 10x = 503.3333... - 50.3333....
90x = 453

양변을 90으로 나누면 x의 값을 알 수 있다.

x = 453/90;
이를 기약분수로 나타내면 151/30 인데,
순환소수의 분수꼴의 분모는 2와 5외의 인수를 가져
유한소수로 나타낼 수 없는 것 또한 증명할 수 있다.
```

4. 루트란?

```
어떤 수 𝑎에 대하여, 𝑥^2 = 𝑎를 만족하는 𝑥를 제곱근(square root)이라 한다.
양의 제곱근만을 특별히 루트(√) 기호로 표시한다.

ex) √5^2 = 5이다.
ex) 5의 제곱근 = +-√5 이다.

루트 전체에 제곱을 할 경우 결과물은 루트가 씌워진 수 그 자체이다.
반대로 루트가 씌워진 수에 제곱을 할 경우 그 결과값은 제곱한 수의 절대값이다.

ex) -√2^2 = -2
ex) √(-2^2) = 2

!이때 -의 제곱이 어떻게 -부호를 달고 나올 수 있는지가 궁금하여 찾아보았는데,
허수 개념이 필요했고, 아직 배우지 않아 추후 학습하여 작성하겠습니다.
```
