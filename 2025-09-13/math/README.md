# [수학] 학습 회고 (2025-09-13)

## 학습한 내용

### 식 변형 공식

1. a^2 + b^2

```
곱셈 공식중 완전제곱식을 응용하면 풀 수 있다.

(a+b)^2 = a^2 + 2ab + b^2

여기서 우변의 2ab를 지우고 좌변에도 똑같이 2ab를 -하면

(a+b)^2 - 2ab = a^2 + b^2
```

2. a^3 + b^3

```
마찬가지로 곱셈 공식을 활용

(a+b)^3 = a^3 + 3ab(a + b) + b^3

(a+b)^3 - 3ab(a + b) = a^3 + b^3
```

3. a^2 + b^2 + c^2

```
(a+b+c)^2 = a^2 + b^2 + c^2 + 2(ab + bc + ca)

(a+b+c)^2 - 2(ab + bc + ca) = a^2 + b^2 + c^2
```

4. (a+b)^2 - (a+b)^2 = 4ab

```
a^2 + 2ab + b^2 - (a^2 - 2ab + b^2) = 4ab
```
